# -AdventureWorks-Executive-Performance-Dashboard-2020---2022-

📌 Project Overview

This project presents an interactive Executive Performance Dashboard developed using Microsoft Excel to analyze the business performance of AdventureWorks from 2020 to 2022.

The dashboard was designed to provide senior management with a clear and comprehensive view of key business metrics, enabling data-driven decision-making through dynamic visualizations and performance indicators.

📊 Key Features
Interactive dashboard with dynamic filtering and drill-down capabilities.
Analysis of revenue, profit, profit margin, and return rate across multiple years.
Customer demographic analysis, including age segmentation.
Product and category performance evaluation.
Regional sales performance analysis across different territories.
Executive-level KPIs for monitoring overall business performance.
🛠️ Data Preparation & Modeling
Power Query
Imported and cleaned data from multiple related tables.
Removed duplicates, handled missing values, and corrected data types.
Created additional calculated fields such as customer full name and age.
Data Modeling
Built a structured data model using Power Pivot.
Established relationships between fact and dimension tables following a star schema design.
DAX Measures

Developed dynamic calculations including:

Total Revenue
Net Profit
Profit Margin (%)
Return Rate (%)
🎯 Business Value

The dashboard transforms raw transactional data into actionable business insights, helping executives monitor company performance, identify trends, evaluate profitability, and support strategic decision-making.
